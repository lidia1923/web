Formularios

Para especificar el uso de un formulario utilizamos la etiqueta form, esta etiqueta tiene 2 propiedades importantes las cuales son:

  • action: Permite especificar hacia donde, se van a enviar los datos del formulario. Puede ser otra página o la misma página
  • method: Permite especificar como se van a enviar los datos, es decir, visibles para el usuario mediante el metodo get o invisibles mediante post.
Propiedades de la etiqueta input - text:
  • Readonly: Se usa para que el campo solo pueda ser visto, no se manipule
  • Disabled: Se usa para marcar el campo como deshabilitado
  • Maxlength: Permite especificar el numero de caracteres permitidos
  • Maxlength: Permite especificar un mensaje dentro del campo, que se borra cuando el usuario escribe algo
